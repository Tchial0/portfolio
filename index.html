About Me

How everything started?

The first time my father gave me a computer I did not sleep for at least 2 days, I got so amazed that I wanted so much to discover how those windows and buttons where created, how folders reacted to my clicks so fast, how those cables were bringing colored lights to the monitor, how the mouse pointer obeyed carefully my movements. After some researches (there was a precious offline version of wikipedia) I found the answer: there was something called Computer Science.

Programming languages timeline

The first programming language I knew about is Assembly it was hard to understand and some time after I knew that there were easier languages. Some friend of mine told me about The C Programming Language and said "If you learn this language I'll present you Visual Basic, and you will be able to create windows and buttons as you want badly." I learned C and dived into Visual Basic using Visual Studio. When I heard there was a C-like language for web development I got into PHP, PHP forced me to meet HTML,CSS and JavaScript. I did dot like web development, Desktop was more interesting, and this interest presented me a powerful language: C#. Yes, C#.
